# Quiz App - CMPT 276 Assignment 1

## Overview
This repository contains the code for a Quiz App developed as part of the CMPT 276 course - Introduction to Software Engineering at Simon Fraser University.

## Tech Stack
- Java Spring
- HTML
- JavaScript
- Render
